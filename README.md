## Sobre
Este repositório será usado para armazenar meus materiais de exercícios e anotações de estudo relacionados a programação.

Os documentos podem ser usados como uma ajuda nos estudos, um material complementar, por exemplo.

## Estrutura
Dentro dos documentos em markdown estão as anotações das aulas. Nos exercícios há comentários feitos por mim.

As pastas estão organizadas por curso, as que não estão categorizadas são de estudos feitos sem um curso em específico, apenas pesquisando.

Tento deixar as anotações minimamente estruturadas e compreensíveis, levando em conta de que é algo que vou ler novamente depois para fins de revisão. Colocarei links externos e referências que uso.
